# Power Generation Forecasting using ANN

## Overview

This project uses an Artificial Neural Network (ANN) built with PyTorch to predict electrical power generation based on operational parameters of a power plant.

## Dataset Features

- Ambient Temperature (AT)
- Exhaust Vacuum (V)
- Ambient Pressure (AP)
- Relative Humidity (RH)

## Target Variable

- Electrical Energy Output (PE)

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- PyTorch

## Workflow

1. Data Preprocessing
2. Train-Test Split
3. Feature Scaling
4. ANN Model Building
5. Model Training
6. Performance Evaluation
